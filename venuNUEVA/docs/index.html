Venu.mx | Multipurpose Html5 Template

Venu.mx is a Multipurpose Html5 Template based on Html5, CSS3, Bootstrap 5 & designed with great attention to details, flexibility and performance. It is ultra professional, smooth and sleek, with a clean modern layout.

Thank you for purchasing Venu.mx Theme template. This documentation will help to know more about template and how is structured and how to use the template features.

Requirements

To be able to use HTML template, you must have the following:

  • Make sure you upload the required assets files/folders listed below:
    • /assets/images - Images and videos Folder
    • /js/vendors - Extra Javacripts libraries Folder
    • /js/main.js - Main Js file
    • /css/vendors - Extra Stylesheets Folder
    • /css/main-LTR.css - Main Stylesheet File
    • /index.html - Index File/Homepage

Installation

Follow the steps below to get started with your Venu.mx Theme Template:

  1. Open the Theme Folder to find all the html Templates Files
  2. You will need to Upload these files to your Website hosting Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • index.html - Required.
    • css folder - Required.
    • js folder - Required.
    • assets folder - Required.

CSS Structure

Here's a list of the stylesheet files I'm using with this template, you can find more information opening each file:

File Name Description
vendors/libraryName.css This is a bunch of cool, fun, and minified files for all external css resources you can found this resources in css/vendors folder.
main-LTR.css This is the main css file, it contains all the styles of the website.
--[LTR] stands for - Left To Right style.
main-RTL.css This is the main css file, it contains all the styles of the website.
--[LTR] stands for - Right To Left style -.

sass CSS Structure

Here's a list of sass CSS files we are using with this template, you can find more information opening each file:

File Name Description
config Folder This folder contain base scss files that defines the configrations varibales such as page direction or themeName .
helpers Folder This folder contain variable file, globals file, mixins file and more file of that nature.
layout Folder This folder contain scss files for pages layout like pages sections.
components Folder This folder contain scss files for global components that used across all pages like buttons for example.
main.scss In these file we @include all partials scss files and compile it into one scss file

Javascript Structure

The js (scripts) files are loaded in footer. I loaded js Library in the bottom because it will increase the site speed and rendering.

The JS files used in this template are:

File Description
jquery-3.x.x.min.js jQuery is a must have library that all other librarieson the project depends on
libraryFileName.js this bring each javascript liberaries we depend on inside our Template.
main.js the main js file to put all js actions in work

Change Fonts

To change the font, Check head inindex.html then in first go to google fonts choose a font and use the generated code and replace with next code. 

                                        <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=jost:400,500,600,700,800">

Next Step : change font family in main-LTR

Change Color

Venu.mx Theme template have a number of included colour schemes that you can switch to at any time. To switch colour schemes, simply replace color code in /_src/scss/1-helpers/varibales.scss with what you need.

Change Favicon

The main favicon is located in assets/images/fav-icon/fav-icon.png replace with your own.

<link href="assets/images/fav-icon/fav-icon.png" rel="icon">

Change logo

Just edit the logo-light.png from the folder assets/images/logo/ or open index.html and you should see the tag there: 

<img src="assets/images/logo/logo-light.png" class="logo-light" alt="logo">

Change Sections backgrounds

You can use the backgorund image using the Following code:

We used a div with the class overlay-photo-image-bg to add background to any section that we need to ad a background image . put your nedded bg images in this folder assets/images/sections-bg-images, and be sure it have the exact name as the one came with the Venu.mx Theme Theme. you can set the path of the image you want as a value of attribute data-bg-img inside the html element.

you can change background opacity from main-LTR or data-bg-opacity inside the html element

you can give background photo the Paralax effect by simply add paralax to overlay-photo-image-bg div
